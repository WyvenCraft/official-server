## Main World (Overworld)

### Description
The Main World is the heart of WyvenCraft - your primary survival and building world. This is where you'll establish your base, build your home, and create your legacy.

### Features
- **Permanent World** - Never resets, your builds are safe forever
- **Land Claims** - Protect your builds with the claim system
- **Custom Terrain** - Enhanced world generation
- **Biome Variety** - All vanilla biomes plus custom ones
- **Villages** - Enhanced NPC villages with custom trades
- **Dungeons** - Custom dungeons with unique loot
- **Structures** - Custom structures to discover

### World Specifications
- **World Size:** 50,000 x 50,000 blocks
- **World Border:** Yes, expandable
- **Spawn Protection:** 100 block radius
- **Difficulty:** Hard
- **Mob Spawning:** Enabled
- **PvP:** Disabled (except in designated areas)

### Points of Interest
- **Spawn Town** - Central hub with shops and services
- **Shopping District** - Player-run economy center
- **Town Hall** - Community gathering space
- **Tutorial Area** - New player guidance
- **Portal Hub** - Fast travel to other worlds
- **Community Farms** - Public resource farms

### Tips
üí° **Main World Tips:**
- Claim your land early to protect your builds
- Build at least 500 blocks from spawn for more space
- Use `/spawn` to return to spawn
- Set your home with `/sethome` at your base
- Explore to find the best building location

## Resource World

### Description
The Resource World is a separate dimension designed for gathering resources. This world resets monthly to provide fresh resources for all players.

### Features
- **Monthly Resets** - Fresh resources every month
- **No Claims** - Free-for-all gathering
- **All Resources** - Every vanilla resource available
- **Enhanced Ores** - Slightly increased ore generation
- **Random Seed** - Different terrain each reset
- **No Griefing Penalty** - Mine freely without worry

### World Specifications
- **World Size:** 20,000 x 20,000 blocks
- **Reset Schedule:** 1st day of each month (00:00 UTC)
- **Difficulty:** Normal
- **PvP:** Disabled
- **Keep Inventory:** Enabled

### Resources Available
- All ores (coal, iron, gold, diamond, emerald, etc.)
- Stone types (granite, diorite, andesite, deepslate)
- Wood from all tree types
- Sand, gravel, clay
- Flowers and plants
- Mob drops from natural spawns

### Important Notes
‚ö†Ô∏è **Warning:**
- DO NOT build permanent structures here
- Anything in this world will be deleted on reset
- Use ender chests to store valuables
- Check reset countdown with `/rw time`
- Get notified 24 hours before reset

### Access
- Use `/warp resource` or `/rw` to teleport
- Random teleport with `/rw tp` for a fresh area
- Return to main world with `/spawn`

## Nether

### Description
The Nether dimension, enhanced with custom features while maintaining its vanilla challenge and atmosphere.

### Features
- **Permanent World** - Never resets
- **Nether Hub** - Fast travel system connecting bases
- **Custom Fortresses** - Enhanced nether fortresses
- **Biome Diversity** - All nether biomes
- **Bastion Remnants** - Custom loot tables
- **Ancient Debris** - Standard netherite availability

### World Specifications
- **World Size:** 10,000 x 10,000 blocks
- **Difficulty:** Hard
- **PvP:** Disabled
- **Mob Spawning:** Enabled

### Nether Hub System
The Nether Hub provides fast travel:
- **8:1 Ratio** - Travel 8 blocks in overworld per 1 in nether
- **Portal Rooms** - Designated areas for portals
- **Highway System** - Safe ice highways to popular areas
- **Portal Linking** - Staff assistance for proper portal linking

### Tips
üí° **Nether Tips:**
- Bring fire resistance potions
- Watch out for ghasts near portals
- Use `/sethome nether` for easy access
- Ice highways are fastest for long-distance travel
- Request a portal room on Discord

## The End

### Description
The End dimension, home to the Ender Dragon and End Cities, with custom enhancements for better gameplay.

### Features
- **Dragon Fight** - Weekly dragon respawns
- **End Cities** - Custom loot and increased spawn rate
- **Shulker Farming** - Renewable shulkers
- **Elytra Access** - Multiple end cities to find elytra
- **End Gateway Network** - Easy access to outer islands
- **Enderman Farms** - Designated farming areas

### World Specifications
- **World Size:** Unlimited outer islands
- **Difficulty:** Hard
- **PvP:** Enabled during dragon fight only
- **Dragon Respawn:** Every Sunday at 20:00 UTC

### Dragon Fight
- **Participation Rewards** - Everyone gets loot
- **Dragon Egg** - Winner takes the egg
- **Experience Boost** - Massive XP drops
- **Announcement** - Server-wide notification
- **Preparation Time** - 15 minutes before fight

### End Cities
- **Increased Spawns** - More cities to explore
- **Custom Loot** - Enhanced treasure
- **Elytras** - Available in ship treasures
- **Shulker Boxes** - Farm shulkers for boxes
- **Chorus Fruit** - Plentiful for farming

### Access
- Portal at spawn
- `/warp end` - Direct teleport to main island
- End gateways for outer islands

## Creative World

### Description
A dedicated creative world for testing builds, practicing, and unleashing your creativity without limits.

### Features
- **Full Creative Mode** - Fly and build freely
- **WorldEdit Access** - Advanced building tools (trusted players)
- **Plot System** - Personal building plots
- **No Resource Cost** - Unlimited blocks
- **Import to Survival** - Submit builds for main world

### World Specifications
- **Plot Size:** 128 x 128 blocks per player
- **Height Limit:** Full build height (Y: -64 to 320)
- **PvP:** Disabled
- **Commands:** Enhanced creative commands

### Plot System
- **Claim Plot:** `/plot claim`
- **Visit Plots:** `/plot visit <player>`
- **Add Friends:** `/plot trust <player>`
- **Plot Chat:** `/plot chat`
- **Multiple Plots:** Unlock with playtime

### Rules
- No inappropriate builds
- No lag machines
- Respect plot boundaries
- Follow server guidelines

## World FAQ

**Q: Which world should I build my base in?**  
A: Always build in the Main World - it never resets.

**Q: Can I claim land in the Resource World?**  
A: No, claims are disabled. The world resets monthly.

**Q: How do I get to the Nether?**  
A: Build a nether portal or use `/warp nether`.

**Q: When does the Resource World reset?**  
A: First day of each month at midnight UTC. Check `/rw time`.

**Q: Are items safe in the End?**  
A: Yes, but be careful during dragon fights when PvP is enabled.

**Q: Can I have a creative plot?**  
A: Yes! Go to creative world and use `/plot claim`.
