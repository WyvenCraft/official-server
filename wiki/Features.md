## Custom Gameplay

## Land Protection

### Claim System
Protect your builds with our advanced claiming system:

- **GriefPrevention** - Claim land to protect it
- **Golden Shovel** - Use to create and modify claims
- **Automatic Protection** - First chest automatically claims area
- **Claim Blocks** - Earn blocks through playtime
- **Subclaims** - Create smaller claims within main claim
- **Trusted Players** - Grant access to specific players

### Protection Features
- **Container Protection** - Lock chests, furnaces, hoppers
- **Entity Protection** - Protect item frames, armor stands
- **Animal Protection** - Your animals are safe
- **Build Protection** - Nobody can modify your claimed land
- **Explosion Protection** - TNT and creepers won't damage claims
- **Fire Protection** - Fire spread disabled in claims

### Trust Levels
| Trust Level | Can Do |
|-------------|--------|
| **Access Trust** | Use buttons, levers, beds |
| **Container Trust** | Access + open containers, kill animals |
| **Build Trust** | Container + place/break blocks |
| **Permission Trust** | Build + manage trust settings |

## Player Shops

### ChestShop
Create your own shops to buy and sell items:

- **Easy Setup** - Place a sign on a chest
- **Buy Shops** - Sell items to other players
- **Sell Shops** - Buy items from other players
- **Admin Shops** - Server shops with unlimited stock
- **Shop Protection** - Automatically protected
- **Transaction Log** - Track all shop activity

### Shop Format
```
Line 1: [Your Username]
Line 2: Amount
Line 3: B [price] : S [price]
Line 4: Item Name
```

Example:
```
PlayerName
64
B 100 : S 50
Diamond
```
This shop buys 64 diamonds for $100 and sells 64 diamonds for $50.

### Auction House
- **24/7 Trading** - Buy and sell even when offline
- **Search Function** - Find items easily
- **Bidding System** - Bid on items
- **Browse Categories** - Filter by item type
- **Listing Fee** - Small fee to list items
- **Expiration** - Listings expire after set time
