## Installation Guide

### Method 1: CurseForge Launcher (Recommended)

1. **Download CurseForge Launcher**
   - Visit [curseforge.com](https://www.curseforge.com/download/app)
   - Download and install the launcher

2. **Install Modpack**
   - Open CurseForge Launcher
   - Go to "Minecraft" section
   - Click "Browse Modpacks"
   - Search for "WyvenCraft Official"
   - Click "Install"

3. **Launch**
   - Click "Play" once installation completes
   - The launcher will download all required files
   - Join the server: `play.wyvencraft.com`

### Method 2: ATLauncher

1. **Download ATLauncher**
   - Visit [atlauncher.com](https://atlauncher.com/downloads)
   - Download and install

2. **Add Modpack**
   - Open ATLauncher
   - Click "Packs" tab
   - Search "WyvenCraft"
   - Click "Install"

3. **Configure**
   - Select modpack version
   - Choose installation location
   - Click "Install"

### Method 3: Manual Installation

1. **Install Forge**
   - Download Forge 1.20.4 from [files.minecraftforge.net](https://files.minecraftforge.net)
   - Run the installer
   - Select "Install Client"

2. **Download Modpack**
   - Download modpack from our [GitHub Releases](https://github.com/WyvenCraft/official-server/releases)
   - Or from our [Discord](https://discord.gg/wyvencraft)

3. **Install Mods**
   - Extract the downloaded file
   - Copy all files to `.minecraft` folder
   - Mods go in `.minecraft/mods`
   - Config files go in `.minecraft/config`

4. **Launch**
   - Open Minecraft Launcher
   - Select Forge profile
   - Click "Play"

## System Requirements

### Minimum Requirements
- **CPU:** Intel Core i5 or AMD equivalent
- **RAM:** 6 GB (4 GB allocated to Minecraft)
- **GPU:** Integrated graphics
- **Storage:** 5 GB free space
- **OS:** Windows 10, macOS 10.12+, Linux

### Recommended Requirements
- **CPU:** Intel Core i7 or AMD Ryzen 5
- **RAM:** 12 GB (6-8 GB allocated to Minecraft)
- **GPU:** NVIDIA GTX 1060 / AMD RX 580 or better
- **Storage:** 10 GB free space (SSD recommended)
- **OS:** Windows 11, macOS 11+, Linux (recent)

### Optimal Requirements
- **CPU:** Intel Core i9 or AMD Ryzen 7/9
- **RAM:** 16+ GB (8-10 GB allocated to Minecraft)
- **GPU:** NVIDIA RTX 3060 / AMD RX 6700 or better
- **Storage:** 15 GB free space on SSD
- **OS:** Latest version

### RAM Allocation Guide
Set these JVM arguments in your launcher:

**4GB RAM:**
```
-Xmx4G -Xms4G
```

**6GB RAM:**
```
-Xmx6G -Xms6G
```

**8GB RAM (Recommended):**
```
-Xmx8G -Xms8G
```

## Troubleshooting

### Common Issues

#### Game Crashes on Launch
**Solutions:**
1. Update Java to latest version
2. Increase RAM allocation
3. Update graphics drivers
4. Disable shaders
5. Remove Optifine temporarily

#### Low FPS / Lag
**Solutions:**
1. Lower render distance to 8-10
2. Disable shaders
3. Set particles to "Minimal"
4. Enable FPS Reducer
5. Allocate more RAM (but not all)
6. Close other programs

#### Cannot Connect to Server
**Solutions:**
1. Verify modpack version matches server
2. Check internet connection
3. Disable VPN if using one
4. Verify server address: `play.wyvencraft.com`
5. Contact staff on Discord

#### Missing Mods Error
**Solutions:**
1. Reinstall modpack through launcher
2. Manually download missing mods
3. Verify mod versions match
4. Clear mod cache and reinstall

#### Recipe Conflicts
**Solutions:**
1. Install Polymorph mod
2. Use JEI to find alternative recipes
3. Check modpack updates

## FAQ

**Q: Do I need the modpack to play?**  
A: No, but it's highly recommended for the best experience. The server works with vanilla clients but you'll miss custom features.

**Q: Can I add my own mods?**  
A: Client-side mods (minimap, cosmetic) are fine. Don't add mods that give unfair advantages or change gameplay.

**Q: How often is the modpack updated?**  
A: We update regularly - check Discord for announcements. Major updates quarterly, minor fixes monthly.

**Q: Will my world work with updates?**  
A: Most updates are compatible. Backup before updating. Major versions may reset certain features.

**Q: Can I use Fabric instead of Forge?**  
A: No, our modpack requires Forge. We may create a Fabric version in the future.

**Q: Is Optifine required?**  
A: Highly recommended but not required. It significantly improves performance.

**Q: Can I run this on a potato PC?**  
A: Minimum specs should work at low settings. Recommended specs for good experience.

**Q: How do I update the modpack?**  
A: Launchers auto-update. Manual installations need to download new version and replace files.

**Q: Are shaders included?**  
A: No, but shader support is included. Download shaders separately.

**Q: Can I play on servers without the modpack?**  
A: Most mods are client-side, so yes. But server-side features won't work on other servers.
