# Contributing to WyvenCraft

Thank you for your interest in contributing to the WyvenCraft Minecraft Server community! This document provides guidelines for reporting issues, suggesting features, and contributing to our server's improvement.

## ğŸ› Reporting Bugs

Before reporting a bug, please:

1. **Search existing issues** to see if the bug has already been reported
2. **Check the wiki** to ensure it's not expected behavior
3. **Test on the latest version** if possible

When reporting a bug, please use our [Bug Report Template](https://github.com/WyvenCraft/official-server/issues/new?template=bug_report.yml) and include:

- **Clear title** describing the issue
- **Detailed description** of what went wrong
- **Steps to reproduce** the bug
- **Expected vs. actual behavior**
- **Screenshots or videos** if applicable
- **Minecraft version** you're using
- **Any error messages** you encountered

### Bug Severity Guidelines

- **Critical:** Server crashes, data loss, or game-breaking issues
- **High:** Major features not working as intended
- **Medium:** Minor features broken or significant inconvenience
- **Low:** Cosmetic issues or minor inconveniences

## ğŸ’¡ Suggesting Features

We love hearing ideas from our community! Before suggesting a feature:

1. **Check existing feature requests** to avoid duplicates
2. **Review the roadmap** (if available) to see if it's already planned
3. **Consider feasibility** and how it would benefit the community

When suggesting a feature, please use our [Feature Request Template](https://github.com/WyvenCraft/official-server/issues/new?template=feature_request.yml) and include:

- **Clear summary** of the feature
- **Problem statement** it would solve
- **Proposed solution** with details
- **Benefits** to the community
- **Examples** from other servers (if applicable)
